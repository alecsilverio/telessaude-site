# Telessaúde – CEM Três Lagoas (MS)

Site informativo sobre como funciona a telessaúde no Centro de Especialidades Médicas (CEM) de Três Lagoas, MS, mostrando o fluxo real do SUS para agendamento de teleconsultas e o funcionamento da telemedicina no CEM.

## Objetivo

- Explicar de forma simples o que é telessaúde.
- Mostrar o passo a passo do fluxo de atendimento:
  - da consulta na UBS até o agendamento pelo SISREG;
  - da chegada ao CEM até a consulta em telemedicina e o retorno.
- Divulgar as especialidades atendidas por telemedicina (neurologia, endocrinologia, psicologia e psiquiatria).

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Hospedagem via GitHub (Git/GitHub para versionamento)

## Estrutura do site

- **Início** – título, chamada principal e destaques da telessaúde.
- **O que é** – explicação do conceito de telessaúde no CEM de Três Lagoas.
- **Como funciona** – fluxo da UBS e SISREG até a teleconsulta e acompanhamento.
- **Benefícios** – vantagens para pacientes e para o SUS.
- **FAQ** – perguntas frequentes.
- **Sobre** – informações sobre o projeto e o Grupo 4 de Telessaúde.
